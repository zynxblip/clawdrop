Common Questions

What's included in the download? +
Everything you need: Node.js v22.13.1 LTS, OpenClaw CLI (latest), and a launcher that auto-configures your environment. The DMG is 211MB and contains the complete runtime.
Do I need technical knowledge? +
Nope. If you can download a file and drag it to your Applications folder, you can use ClawDrop. The first launch walks you through adding your API keys with a simple dialog.
Is it really offline-capable? +
Yes. Once downloaded, the entire setup works without internet. Of course, your AI agent will need connectivity to talk to APIs (OpenAI, etc.), but the ClawDrop installation itself is fully self-contained.
How do I update? +
ClawDrop checks for updates automatically and notifies you. Download the new DMG, replace the app in Applications, done. Your configs and data stay intact.
Can I build from source? +
Absolutely. The build script is on GitHub. It downloads Node.js, installs OpenClaw, bundles everything, and creates the DMG. Fully transparent, fully auditable.
